Getting Started (Quickstart)

The easiest way to get started with the Apicurio Studio is to download the quickstart (based on Tomcat 8) and fire it up! All you need is to have Java (version 8) installed, and you're ready to rock.

Download & Install

  1. Apicurio Studio {{ latestRelease.tag_name }}
  2. Unpack the Apicurio Studio Quickstart zip

Run

  1. Start Tomcat
  2. Point your browser at Apicurio Studio and login with your GitHub credentials!

Configure

The quickstart distribution comes pre-configured with everything you need to get started. There is no additional configuration for you to perform!

Or simply try this...

mkdir ~/apicurio-studio-{{ site.version }}
cd ~/apicurio-studio-{{ site.version }}
curl -L {{ theasset.browser_download_url }} -o {{ theasset.name }}
unzip {{ theasset.name }}
cd apicurio-studio-{{ site.version }}
./bin/catalina.sh run

Getting Started (Docker)

If you prefer to get started by using the Apicurio Studio docker image, that's cool too! Just follow the instructions in this section instead.

Download

  1. Use docker to pull the Apicurio Studio docker image: apicurio/apicurio-studio

Run

  1. Use docker to run the downloaded image 

    docker run -it -p 8080:8080 apicurio/apicurio-studio

  2. Point your browser at Apicurio Studio and login with your GitHub credentials!

Configure

Apicurio Studio comes pre-configured with everything you need to get started. There is no additional configuration for you to perform!

Or simply try this...

docker pull apicurio/apicurio-studio:latest
docker run -it -p 8080:8080 -p 8443:8443 apicurio/apicurio-studio:latest
