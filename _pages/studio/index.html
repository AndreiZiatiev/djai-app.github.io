What is Apicurio Studio?

Apicurio Studio is a web based API Design suite, primarily (but not exclusively) built to support design-first REST API development. It provides a way for users to collaboratively and visually design their APIs prior to (or concurrent with) the implementation.

Key Features

Web Based
Design your OpenAPI APIs in a web (browser based) application - no installation necessary.
Code-optional Design
Don't know the OpenAPI specification inside and out? Now you don't have to!
Open Source
The Apicurio Studio is fully open source, hosted on GitHub and actively maintained.
